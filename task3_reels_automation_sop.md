# 任務3｜Reels 短影音排程 (IG月華星礦坊)

## 1. 任務核心設定
- **發布時間**：每週一、四 18:00 HKT
- **目標**：90天內發布 25+ 則 Reels
- **核心策略**：高完播率、風水礦石主題、一千零一夜「連載式懸念」。
- **三語規範**：
  - **廣東話 VO**：口語、生動、神秘張力（<300字）。
  - **正體中文字幕**：書面語，港台友好。
  - **英文字幕**：精簡自然。

## 2. 影片規格
- **長度**：嚴格 < 30 秒。
- **格式**：AI 繪圖動畫 + 配音。

## 3. 連載主題架構 (90天25集)
- **集數 1-8**：破圈引流 — 禁忌與誤區系列。
- **集數 9-16**：信任建立 — 玄學科普系列。
- **集數 17-25**：產品轉化 — 場景應用系列。

## 4. 排程邏輯 (Automation)
每逢週一、四 18:00 HKT，系統將調用 `create_instagram` 工具發布內容。
- **排程指令範例**：
```bash
manus-config schedule create \
  --title "Reels Task 3: Episode [X]" \
  --detail "發布第 [X] 集 Reels：[標題]。包含三語字幕與廣東話 VO 動畫。" \
  --cron "0 0 18 * * 1,4" \
  --repeated
```

## 5. 腳本庫 (Scripts Library)
### 第 1 集：呢3種水晶千萬唔好曬太陽！
- **Hook (0-3s)**：等等！你日日曬太陽嘅水晶，其實係廢咗㗎！
- **展開 (3-15s)**：紫水晶、粉晶、海藍寶——呢三種礦石含鐵同錳，一見太陽就褪色。
- **戲肉 (15-25s)**：仲有一種，幾乎每個人都有，佢一曬太陽唔止褪色，仲會……（黑屏懸念）
- **CTA (25-30s)**：追蹤我，第 2 集答你！
