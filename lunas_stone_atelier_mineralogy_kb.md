## 1. 科學定義與基礎語言 (Scientific Terminology)

為了確保品牌的專業度，所有對外溝通的「天然礦石」定義必須基於以下科學標準語言：

- **礦物 (Mineral)：** 在地殼中自然形成、成分與結構相對穩定、具有晶體結構的無機固體（例如：石英 SiO₂、長石、方解石等）。
- **岩石 (Rock)：** 由多種礦物聚合而成（例如：花崗岩由石英、長石、雲母組成；青金石為多礦物岩石）。
- **寶石 / 水晶級礦物：** 顏色美、透明度高、耐久稀有，經切磨後可作飾品（如石英族水晶、寶石級尖晶石、黃玉、剛玉等）。
- **天然礦石範疇：** 雖然目前已知天然礦物超過 5,000 種，但真正常見於珠寶、水晶療癒與市售飾品領域的，大約只有數十到百餘種。

### 1.1 外觀分辨原則（天然 vs 人工/優化）

在描述如何辨別礦石時，必須遵循以下視覺特徵指標：

**天然礦石常見特徵：**
- **內含物多樣：** 棉絮狀、針狀、雲霧、色帶、晶面生長紋等，自然不規則。
- **顏色有深淺與分區：** 常有直線或三角形色帶，非完全均一。
- **光澤與火彩有層次：** 火彩深邃，不會像玻璃那樣單一死板。
- **形狀與裂紋自然：** 打磨後仍可見微細裂理或紋理。

**人工優化/合成或仿品特徵（警訊）：**
- **顏色過於均勻或具「螢光感」：** 例如整顆粉晶通透無雲霧、有色瑪瑙顏色高度一致（可能染色或合成）。
- **幾乎沒有內含物：** 在 10 倍放大鏡下非常乾淨（對合成剛玉等品種反而是警訊）。
- **氣泡：** 玻璃仿製品常有圓形氣泡。
- **過輕或過重：** 玻璃/樹脂常重量偏低，部分合成素材偏重。

> **免責聲明標準句：** 實務上「只靠肉眼」難做到百分百判定，真正的確定方式仍需依賴專業寶石鑑定證書（如 GIA、GRS 或當地公會實驗室）。

### 2.2 核心礦石標準紀錄 (Standard Records)

#### 1. 白水晶 (Clear Quartz)
- **分類 / 晶系：** 石英族 / 六方晶系
- **化學式 / 硬度：** SiO₂ / 莫氏硬度 7
- **外觀特徵：** 透明～半透明無色，常見冰裂紋、雲霧狀內含物，玻璃光澤強。
- **對應脈輪：** 頂輪（主要）、全脈輪平衡。
- **能量主題：** 放大能量、淨化、平衡全身，像通用放大鏡與訊號轉接器。協助專注、清晰、提升靈性覺察。
- **保養與淨化：** 可用白水晶簇、月光、煙薰、聲音、短暫清水沖洗。可短暫日曬，不宜超長時間曝曬。

#### 2. 紫水晶 (Amethyst)
- **分類 / 晶系：** 石英族 / 六方晶系
- **化學式 / 硬度：** SiO₂（含鐵致色） / 莫氏硬度 7
- **外觀特徵：** 淺紫至深紫，色帶常呈不均勻分布，內含雲霧與細裂紋常見。
- **對應脈輪：** 眉心輪（主要）、頂輪。
- **能量主題：** 鎮定心神、助睡眠、提升靈性與直覺，支持冥想與戒除依賴。
- **保養與淨化：** 可用白水晶簇、月光、煙薰、聲音。**忌長時間曝曬（易褪色）**。

#### 3. 粉晶 (Rose Quartz)
- **分類 / 晶系：** 石英族 / 六方晶系（多為塊狀）
- **化學式 / 硬度：** SiO₂（含 Ti, Fe, Mn 微量） / 莫氏硬度 7
- **外觀特徵：** 淺粉～中粉，多為半透明至霧面朦朧感，雲霧狀、棉絮狀內含物常見。
- **對應脈輪：** 心輪。
- **能量主題：** 自我接納、柔軟心輪、修復感情創傷，提升溫柔與共感。
- **保養與淨化：** 可用白水晶簇、月光、煙薰。**忌長時間日曬（易退色變白）**。

#### 4. 茶晶 (Smoky Quartz)
- **分類 / 晶系：** 石英族 / 六方晶系
- **化學式 / 硬度：** SiO₂（輻射致色） / 莫氏硬度 7
- **外觀特徵：** 淺棕、煙茶色、深棕咖啡色，顏色多有濃淡變化。
- **對應脈輪：** 海底輪（主要）、太陽神經叢。
- **能量主題：** 將負面能量下導至大地，穩定情緒、提升踏實感與實踐力。
- **保養與淨化：** 可用白水晶簇、月光、煙薰、短暫水洗。不宜長時間烈日曝曬。

#### 5. 黃水晶 (Citrine)
- **分類 / 晶系：** 石英族 / 六方晶系
- **化學式 / 硬度：** SiO₂（含鐵致色） / 莫氏硬度 7
- **外觀特徵：** 淺黃、金黃、橙黃，色帶與深淺不均常見。若顏色偏橙褐且根部白，多為高溫燒色紫水晶。
- **對應脈輪：** 太陽神經叢（主要）、臍輪。
- **能量主題：** 自信、行動力、財富顯化與目標實現，幫助理性思考與決策。
- **保養與淨化：** 可用白水晶簇、月光、煙薰。**忌長時間曝曬（易褪色或改變色澤）**。

#### 6. 拉長石 (Labradorite)
- **分類 / 晶系：** 斜長石類 / 三斜晶系
- **化學式 / 硬度：** (Ca,Na)(Al,Si)₄O₈ / 莫氏硬度 6–6.5
- **外觀特徵：** 灰黑至深藍底色，具拉長石效應（Labradorescence），有藍、綠、金等彩色閃光。
- **對應脈輪：** 眉心輪（主要）、頂輪。
- **能量主題：** 靈性保護、洞察潛意識、啟動天賦與直覺，適合靈性工作者。
- **保養與淨化：** 可用白水晶簇、月光、煙薰、聲音。建議少水洗（有裂理避免溫差），中度避免長曝。

#### 7. 青金石 (Lapis Lazuli)
- **分類 / 晶系：** 多礦物岩石（以天藍石為主） / 等軸晶系
- **化學式 / 硬度：** Na₈(AlSiO₄)₆(SO₄,S,Cl)₂（含方解石與黃鐵礦） / 莫氏硬度 5–5.5
- **外觀特徵：** 深藍底色，常帶黃鐵礦小金點與白色方解石斑，星空感。
- **對應脈輪：** 眉心輪（主要）、喉輪。
- **能量主題：** 提升專注與思考清晰，協助表達真實想法與靈性洞察。
- **保養與淨化：** 可用白水晶簇、月光、煙薰。**忌水洗（多孔含方解石黃鐵礦易損傷）**，避免長曝。

#### 8. 黑曜石 (Obsidian)
- **分類 / 晶系：** 火山玻璃 / 無定形
- **化學式 / 硬度：** 富矽火山玻璃 / 莫氏硬度 5–5.5
- **外觀特徵：** 黑色、墨綠，斷口貝殼狀，部分有彩虹光暈（彩虹黑曜）或金光（金曜石）。
- **對應脈輪：** 海底輪。
- **能量主題：** 強力吸收與防護負面能量，幫助看見陰影與潛意識。

#### 9. 黑碧璽 (Black Tourmaline)
- **分類 / 晶系：** 碧璽族 / 三方晶系
- **化學式 / 硬度：** 複雜硼矽酸鹽 / 莫氏硬度 7–7.5
- **外觀特徵：** 黑色不透明，柱狀平行條紋明顯，縱向溝槽。
- **對應脈輪：** 海底輪。
- **能量主題：** 穩定、接地、防護電磁與情緒干擾，強化邊界感。

#### 10. 捷克隕石 (Moldavite)
- **分類 / 晶系：** 隕石玻璃（撞擊玻璃） / 無定形
- **化學式 / 硬度：** 矽酸鹽玻璃 / 莫氏硬度 5.5–6
- **外觀特徵：** 深綠色透明至半透明，表面常具蝕刻紋與自然坑洞。
- **對應脈輪：** 心輪（主要）、眉心輪、頂輪。

## 3. 日常保養與淨化規範 (Care & Cleansing Protocol)

### 3.1 淨化方式優先順序
**多數礦石適用的溫和方式：**
1. **白水晶/晶簇淨化：** 放在大塊白水晶簇或晶洞上數小時至一晚。
2. **月光照射：** 放於窗邊接受月光（滿月尤佳）數小時。
3. **薰香/煙霧淨化：** 使用鼠尾草、聖木、線香等繞行。
4. **頌缽/音叉/誦咒：** 利用聲波重置能量，對脆弱礦石極安全。

### 3.2 嚴格禁忌清單
**忌日曬的礦石（易褪色/乾裂）：**
- 紫水晶、粉晶、黃水晶、月光石、蛋白石、染色瑪瑙/水晶。
- *建議：若需日曬淨化，限 10–20 分鐘，選早晨或傍晚斜射光。*

**忌水洗或長時間浸泡的礦石（易溶解/生鏽/失去光澤）：**
- **青金石：** 含方解石與黃鐵礦，易變色裂縫。
- **天青石：** 極易溶於水。
- **蛋白石：** 含水，水分變化易裂或失去變彩。
- **綠松石：** 多孔易吸水變色。
- **孔雀石：** 含銅，遇水及汗水易腐蝕（粉塵有毒）。
- **黃鐵礦群/赤鐵礦：** 長期潮濕易生鏽。

### 3.3 實務保養指引 (SOP)
- **使用後擦拭：** 佩戴後用軟布輕拭汗水油脂。
- **避免劇烈碰撞：** 硬度高不代表不易碎（如月光石、黃玉、磷灰石有裂理）。
- **避免化學品：** 遠離香水、清潔劑、泳池氯水。
- **分開收納：** 硬度高的（石英、剛玉）與較軟礦物分開。
- **淨化頻率：** 高頻率佩戴者 1–3 天輕度淨化；偶爾佩戴者每次使用後淨化。
