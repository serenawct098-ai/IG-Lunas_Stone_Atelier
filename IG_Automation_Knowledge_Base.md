## 3. 核心礦石資料庫 (Core Mineral Database)

本資料庫為礦物介紹、能量解說與保養指引的**唯一真值源**。

### 3.1 脈輪對應總覽
| 脈輪 | 核心主題 | 常見對應礦石（舉例） |
|------|----------|----------------------|
| 海底輪 | 安全感、地氣、行動力 | 黑曜石、黑碧璽、茶晶 |
| 臍輪 | 情感、創造力、慾望 | 黃水晶、蛋白石 |
| 太陽神經叢 | 自信、意志、行動決策 | 黃水晶、虎眼石 |
| 心輪 | 愛、關係、寬恕 | 粉晶、捷克隕石、翡翠 |
| 喉輪 | 溝通、表達、真實 | 海藍寶、天河石 |
| 眉心輪 | 直覺、洞察、靈視 | 紫水晶、拉長石、青金石 |
| 頂輪 | 靈性、超意識、連結 | 白水晶、紫水晶 |

### 3.2 礦石標準紀錄 (Standard Records)

#### 白水晶 (Clear Quartz)
- **化學式/硬度：** SiO₂ / 莫氏硬度 7
- **對應脈輪：** 頂輪（主要）、全脈輪平衡
- **保養與淨化：** 可短暫日曬，不宜超長時間曝曬。

#### 紫水晶 (Amethyst)
- **化學式/硬度：** SiO₂（含鐵致色） / 莫氏硬度 7
- **對應脈輪：** 眉心輪（主要）、頂輪
- **保養與淨化：** **忌長時間曝曬（易褪色）**。

#### 粉晶 (Rose Quartz)
- **化學式/硬度：** SiO₂ / 莫氏硬度 7
- **對應脈輪：** 心輪
- **保養與淨化：** **忌長時間日曬（易退色變白）**。

#### 青金石 (Lapis Lazuli)
- **化學式/硬度：** 多礦物岩石 / 莫氏硬度 5–5.5
- **對應脈輪：** 眉心輪（主要）、喉輪
- **保養與淨化：** **忌水洗（多孔含方解石黃鐵礦易損傷）**，避免長曝。

*(註：其他礦石如茶晶、黃水晶、拉長石、黑曜石、黑碧璽、捷克隕石、海藍寶、翡翠等，均需嚴格參照原始知識庫數據)*

### 3.3 日常保養與淨化規範
- **推薦淨化方式：** 白水晶簇、月光照射、薰香/煙霧、頌缽/音叉。
- **嚴格禁忌：**
  - **忌日曬：** 紫水晶、粉晶、黃水晶、月光石、蛋白石。
  - **忌水洗：** 青金石、天青石、蛋白石、綠松石、孔雀石。

## 4. 銷售與文案話術框架 (Copywriting Framework)

所有生成的 IG 貼文、Reels 文案，必須遵循以下三層結構（科學 → 美學 → 能量）：

2. **美學亮點：** 它最迷人的地方，是那種 `{顏色}` 的色澤，加上 `{天然特徵/光學效應}`，每一顆的紋理與光影都不完全一樣，展現天然礦石的生命感。
3. **天然鑑定：** 挑選時特別看 `{天然特徵}`，避免顏色太均勻、像染色玻璃的質感。
4. **能量與心理：** 能量上對應 `{脈輪}`，主題是 `{能量主題}`。若你在面對 `{適用情境}`，這顆石頭會像一個溫柔的教練/護盾，提醒你回到核心狀態。
5. **保養提醒：** 保養上記得 `{避水/避曬警告}`，平常可用 `{淨化方式}` 幫它定期淨化。
